# EduSwap

EduSwap is a campus marketplace platform designed for students to buy, sell, and exchange books, notes, electronics, and daily essentials within their college community.

## Features

* User Authentication
* Product Listings
* Category Filtering
* Responsive Design
* Marketplace Dashboard
* Student-Focused Marketplace

## Tech Stack

* React.js
* JavaScript
* Tailwind CSS
* FastAPI
* MongoDB

## Installation

```bash
npm install
npm start
```
